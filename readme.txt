A simple before and after image block for photographers.

== Description ==

This is a simple befor and after image block plugin for photographers. Add the block, select the main image, and then select the background image in the toolbar.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/before-after` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

= Does this work in the block editor? =

Yes, this is a block plugin.

= Does it work on mobile browsers? =

Yes, though sometimes the user will have to "click" outside of the image to reset the before image.
